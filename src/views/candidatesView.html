Conectar a Base de Datos

Conecta una base de datos externa (Supabase, API REST, etc.) para importar candidatos automáticamente.

En Netlify se usa el endpoint `/.netlify/edge-functions/supabase-proxy`.
Fuera de Netlify puedes pegar `https://tu-proyecto.supabase.co` para conexión directa.

Opcional si ya definiste `VITE_SUPABASE_ANON_KEY` en tu entorno local.
